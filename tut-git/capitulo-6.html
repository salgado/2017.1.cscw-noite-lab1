Capitulo 6


Git Push

  • Depois de clonar o projeto, efetuar as alterações e efetuar o commit, chegou a hora de enviar para o repositório, para isso, iremos utilizar o comando git push.

  • Execute os comandos abaixo.

  • Ex:

  • Clone o projeto do repositório github:
  • git clone https://github.com/salgado/2017.1.cscw-noite-lab1.git

  • Entre na pasta do projeto, crie uma pasta chamada aula_git e acesse a pasta:

  • cd 2017.1.cscw-noite-lab1

  • mkdir aula_git

  • cd aula_git

  • Agora do tipo texto com seu nome, abra-o, digite seu nome e salve:

  • touch elson.txt

  • gedit elson.txt

  • Chegou a hora de enviar o arquivo criado para o repositório, para isso devemos seguir os seguintes passos:

  • git add elson.txt

  • git commit -m "Inclusão de arquivo para aula git"

  • Antes de enviar para o servidor, você deve atualizar o projeto em sua maquina:

  • git pull

  • Agora envie suas alterações para o repositório:

  • git push

  • Caso tenho seguido os passos corretamente, o arquivo criado agora estará disponível no repositório.
