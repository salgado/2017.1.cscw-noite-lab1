Clonando um repositório

Para clonar um repositório acesse a página do projeto que deseja clonar como exemplo https://github.com/salgado/2017.1.cscw-noite-lab1 . Clique na opção clonar e copie o link que será exibido conforme a imagem abaixo.

Abra o terminal navege até a pasta onde você deseja clonar o repositório e escreva o comando:
git clone <O link obtido no passo acima>

Aguarde até que o download seja feito e pronto seu repositório foi clonado
